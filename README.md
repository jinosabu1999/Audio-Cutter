# Audio Cutter App

![Audio Cutter App](https://telegra.ph/file/fedcc5928a6b1e95b4ecc.jpg)

## Description

A simple web application that allows users to upload an audio file, select a start and end point, and then download the cropped audio. The app uses the Web Audio API to perform the audio cutting operation.

## Features

- Upload audio files in various formats (e.g., MP3, WAV)
- Set start and end points using a time picker
- Download the cropped audio file in MP3 format

## Usage

1. Clone the repository: `git clone https://github.com/jinosabu1999/audio-cutter-app.git`
2. Open the `index.html` file in your browser
3. Upload an audio file, set the start and end points, and click "Cut the Audio"
4. Download the cropped audio file

## Technologies Used

- HTML5
- CSS3
- JavaScript (Web Audio API)
